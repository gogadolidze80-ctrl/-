# RC Batumi — რაგბის საიტი

React + Vite-ზე აგებული RC Batumi-ს ოფიციალური საიტი.

## გაშვება

```bash
npm install
npm run dev
```

ბრauzeri: `http://localhost:5173`

## ბuild

```bash
npm run build
npm run preview
```

## სტრუქტურა

- `public/logo.jpg` — RC Batumi-ს ლოგო
- `src/components/` — React კომპონენტები
- `src/data/siteData.js` — მონაცემები (ცხრილი, მატჩები, სიახლეები)
